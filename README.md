# AI Developer Learning Journey

This repository is for learning Python, LLMs, LangChain, and AI agent development step by step.

## Goal

The goal is to become an AI developer by building strong Python fundamentals first, then moving into APIs, prompt engineering, LLM concepts, LangChain, and small AI projects.

## Learning Path

### 1. Python Basics
- Variables and data types
- Lists and dictionaries
- Conditions and comparisons
- Loops
- Functions

### 2. Intermediate Python
- File handling
- JSON and CSV
- Error handling
- Virtual environments
- Working with APIs

### 3. LLM Fundamentals
- What LLMs are
- Tokens and context windows
- Prompt design
- Structured output
- Basic model evaluation

### 4. AI Frameworks
- LangChain basics
- LangGraph basics
- Tool calling
- Retrieval-Augmented Generation (RAG)
- Memory and workflows

### 5. Projects
- Prompt-based summarizer
- Simple chatbot
- Tool-using assistant
- Document Q&A bot
- Mini AI agent project

## Daily Study Plan

A simple 1-hour daily structure:
- 15 minutes review
- 20 minutes learning new concepts
- 20 minutes coding practice
- 5 minutes notes and reflection
